OpenOA Examples¶

The ENTR Runtime includes example analysis notebooks that demonstrate operational wind plant data analytics use cases using OpenOA with example data stored in the ENTR warehouse. The example notebooks are located at /examples in the ENTR OpenOA repository. All examples use two years of data for the 4-turbine “La Haute Borne” wind plant.

Running the Examples¶

  1. Complete the Installation section and open Jupyter Hub in your web browser.

  2. On the left-hand side of Jupter Hub, navigate to: src/OpenOA/examples/entr

  3. Double click on any example notebook to open it.

List of Examples¶

The ENTR runtime contains two example notebooks:

  • Example 1: Basic toolkit functions

  • Example 2: Long-term AEP assessment

OpenOA documentation¶

OpenOA documentation is hosted on ReadTheDocs.

Data are stored and organized in OpenOA using a PlantData object. The PlantData class in the OpenOA fork included in the ENTR runtime contains a from_entr method (link to source code with documentation), which is used to load data into OpenOA from the ENTR warehouse.
